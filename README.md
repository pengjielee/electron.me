## iboard

我的粘贴板，主要实现把系统剪贴板中的内容持久存储。

使用方法：
1. 按Command + C 复制到系统粘贴板；
2. 按Command + Option + S 保存到我的粘贴板；

实现功能：
1. 支持按日期筛选；
2. 支持按日期/页码导出；
3. 支持导出txt/json文件；

## ikms

我的知识管理系统，Electron + Vue实现，存储使用sqlite数据库；

实现功能：
1. 管理Url；
2. 管理代码；
3. 管理笔记；
4. 设置（换肤、导入、导出、备份）；
